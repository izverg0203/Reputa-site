# 13-ui-components.md  
## UI Components — Компонентная система Reputa 2.0 (Next.js + shadcn/ui)

Документ описывает **все UI-компоненты**, которые используются на сайте Reputa.  
Каждый компонент включает:

- назначение  
- обязательные пропсы  
- вариации (если есть)  
- требования к анимации  
- требования к адаптиву  
- связи с другими компонентами  

Основная задача — создать **единый компонентный язык**, чтобы любая доработка была прогнозируемой.

---

# 1. Core Components (Базовые)

## 1.1. `Button`
Используется для всех CTA.

Варианты:
- primary (синий акцент)
- secondary (серый)
- outline
- ghost
- full-width (mobile)

Motion:
- hover scale 1.03  
- focus ring  

---

## 1.2. `Input`
Используется в формах.

Поддерживает:
- text / email / phone  
- validation state  
- icons-left  

---

## 1.3. `Textarea`
Применение:
- ответы на отзывы  
- AI-ассистент  

---

## 1.4. `Card`
Базовый контейнер всех секций.

Типы:
- elevated  
- outlined  
- glass (только AI-блоки)

---

## 1.5. `Section`
Структурный компонент страницы.

Пропсы:
- title  
- subtitle  
- children  
- theme (light / dark)  

---

## 1.6. `Badge`
Используется для:
- sentiment (Positive / Negative / Critical)  
- категории (price / service / wait-time)  
- новинка / подсказка  

Цветовая логика:
- Critical — красный  
- Negative — оранжевый  
- Positive — зелёный  
- Neutral — серый

# 2. Навигация

- меняет прозрачность при скролле

Содержит:
- ссылки  
- CTA

---

## 2.3. `Footer`

1. маркетинговый

# 4. AI Components

## 4.1. `AIAssistantBubble`
Кнопка, которая открывает чат.

Motion:
- bounce 3% при появлении  
- glowing ring (опционально)  

---

## 4.2. `AIAssistantWindow`
Модальный чат.

Содержит:
- логи диалога  
- индикация typing  
- input  

---

## 4.3. `AIInsightCard`
Используется в секции “AI-инсайты”.

Содержит:
- иконку  
- заголовок  
- краткий инсайт  
- категорию  

---

## 4.4. `AIMetric`
Компонент цифр/метрик:
- рост  
- улучшение тональности  
- динамика
