# 03-design-guidelines.md  
## Design Guidelines — визуальная система бренда Reputa

---

# 1. Эмоциональная основа дизайна

Reputa должна вызывать ощущение:

### **«Тёплая технологичность + защита + рост»**

Это три ключевых эмоциональных направления:

1. **Технологичность**  
   - чистые линии  
   - спокойные AI-эффекты  
   - динамика при скролле  
   - ощущение умной системы

2. **Теплота и человечность**  
   - мягкие тени  
   - округлые элементы  
   - дружелюбные CTA  
   - световые акценты

3. **Рост и движение вперёд**  
   - графики, тренды  
   - элементы, показывающие улучшение  
   - цветовые акценты роста (зелёно-бирюзовые)

Эти три направления должны быть заметны на **каждом экране**, включая лендос, дашборд, AI-помощника, PDF-отчёты.

---

# 2. Цветовая система

Reputa использует **тёплую, но технологичную палитру**, без холодного токсичного синего.

### **Основная палитра**

| Назначение | Цвет | Hex |
|-----------|------|------|
| Primary | Бирюзово-синий | `#3A88F7` |
| Secondary | Тёплый бирюзовый | `#41D3BD` |
| Accent | Лаймовый акцент роста | `#A3FF93` |
| Background Light | Белый | `#FFFFFF` |
| Background Dark | Глубокий тёмно-синий | `#0D1B2A` |
| Text Primary | Тёмный графит | `#1C1C1E` |
| Text Secondary | Серый | `#7A7A7D` |
| Critical | Красный | `#FF4D4D` |
| Positive | Зелёный | `#2FDB8C` |

### Принципы использования:
- **Primary = кнопки, AI-метки, ссылки**
- **Secondary = элементы динамики (графики, подсветки)**  
- **Accent = акценты роста, позитивные тренды**
- **Dark Mode блоки = AI-анализ, технические части**
- **Critical = только для негативных отзывов и предупреждений**

---

# 3. Типографика

Reputa — это сочетание «умной» и «тёплой» типографики.

### Основные шрифты:
- **H1–H2:** Inter / Manrope (SemiBold)  
- **H3–H6:** Inter / Manrope (Medium)  
- **Body:** Inter (Regular)  
- **Numbers, графики:** SF Pro Display / Inter (Medium)

### Типографическое настроение:
- крупные, уверенные заголовки  
- комфортная межстрочная высота (1.4–1.6)  
- чистые интервалы  
- *минимум декоративности, максимум ясности*

---

# 4. Компоненты UI

## Округлая кнопка (главный элемент бренда)
Стиль:  
- сильный овал  
- градиентное свечение  
- лёгкая тень  
- внутри — иконка + текст  
- может быть плавающей кнопкой (как чат-помощник)

## Карточки
- мягкие тени  
- 12–16px скругление  
- тёплая подсветка на hover  
- лаконичные, без перегрева информацией

## Тёмный AI-блок
Используется для:
- AI-анализ  
- распределение тональности  
- топ проблем

Цвет: `#0D1B2A`  
Акценты: бирюзовый, яркие точки трендов.

---

# 5. Motion / Анимации

Анимации — ключевая часть бренда Reputa.  
Они создают ощущение «умного живого продукта».

### Принципы:
- **длительность:** 250–380 ms  
- **easing:** ease-out / cubic-bezier  
- **цель:** поддерживать историю, не отвлекать  
- **стиль:** мягкие параллаксы, подвижные графики, пульсация AI-меток

### Где должны быть анимации:
- Hero-блок (диаграммы оживают)  
- AI-анализ в тёмном блоке  
- кнопка AI-помощника  
- карточки отзывов при появлении  
- плавная подсветка тегов "Critical" / "Positive"

### Где НЕ должно быть анимаций:
- формовые элементы  
- платежи  
- таблицы с KPI

---

# 6. AI-помощник (UI + UX)

AI-помощник — часть бренда. Он должен:

- быть круглым  
- мягким  
- с анимацией дыхания  
- менять выражение статусом (online / thinking)

### Взаимодействие:
- приветствует тепло  
- объясняет пользу  
- делает мини-аудит  
- может предлагать получить PDF-отчёт  
- НЕ должен пушить «оставьте контакты» (только после диалога)

---

# 7. Принципы компоновки страниц

### 1. Блоки — крупные, дышащие  
Без плотной сетки, как у российских «маркетинговых» сайтов.

### 2. Главный ритм — чередование светлых и тёмных секций  
Это создаёт премиальный вид.

### 3. Всегда есть элемент «движения»  
- график  
- диаграмма  
- всплывающие теги  
- пульсация AI-инсайтов  

### 4. CTA — всегда заметные  
Основные CTA:  
- «Получить AI-аудит»  
- «Записаться на демо»

---

# 8. Особые требования к адаптиву

### Мобильный вид — приоритет
- меньше текста  
- больше акцентов  
- CTA всегда на виду  
- AI-помощник фиксирован справа снизу  

### На планшете:
- двухколоночные макеты  
- меньше горизонтальных графиков  

---

# 9. Ошибки, которых нужно избежать

- слишком холодный или медицинский стиль  
- перегрузка текста  
- отсутствие динамики  
- статичные графики  
- однообразие блоков  
- навязчивый чат  

---

# 10. Финальная визуальная формула Reputa

**Reputa =  
чистая технологичность  
+ мягкие интерфейсы  
+ динамика роста  
+ уверенная, спокойная атмосфера  
+ AI, который помогает, а не пугает.**
